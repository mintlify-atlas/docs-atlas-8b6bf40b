Neo Letter is building a global leaderboard to surface the top players across all games. The leaderboard is intended to rank players by their `totalPoints` stat — the running total of every point earned since account creation. The page already exists at `/leaderboard`, but data is not yet fetched or displayed.

<Info>
  The leaderboard is in active development. The underlying Firestore query is defined but not yet subscribed to, so the list on `/leaderboard` is currently empty. This page will be updated when the feature is complete.
</Info>

## What the leaderboard will show

When the leaderboard is fully implemented, it will display the top 10 players ordered by total points. Each entry will show the player's username and their position in the ranking.

| Column | Description |
|---|---|
| **Rank** | The player's position (1st, 2nd, 3rd, …, 10th) |
| **Username** | The player's chosen Neo Letter username |
| **Total Points** | Cumulative points earned across all games |
